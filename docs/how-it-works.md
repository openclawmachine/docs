# How It Works

## Today

1. The user supplies a direct PriceCharting product page.
2. OpenClawMachine refreshes or reads the stored market snapshot for that product.
3. The agent compares the asking price against those comps.
4. The user decides whether to proceed manually.

## Later

1. The user requests a pack or listing purchase.
2. The agent funds and executes the buy flow safely.
3. The asset is delivered.
4. Fulfillment receipts and wallet attestations are returned.

The second flow is intentionally deferred.
